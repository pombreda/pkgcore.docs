filter-env¶

synopsis¶

filter-env [-h] [–version] [–debug] [-V] [-F] [–input INPUT] [–funcs FUNCS] [–vars VARS] [–print-vars]

optional arguments¶

-h, --help show this help message and exit
--version show program’s version number and exit
--debug Enable debugging checks
-V, --var-match
 Invert the filtering- instead of removing a var if it matches remove all vars that do not match
-F, --func-match
 Invert the filtering- instead of removing a function if it matches remove all functions that do not match
--input INPUT, -i INPUT
 Filename to read the env from (uses stdin if omitted).
--funcs FUNCS, -f FUNCS
 comma seperated list of regexes to match function names against for filtering
--vars VARS, -v VARS
 comma seperated list of regexes to match variable names against for filtering
--print-vars print just the global scope environment variables that match
