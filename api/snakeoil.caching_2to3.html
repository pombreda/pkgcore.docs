core caching logic

Roughly, this works by intercepting 2to3 converter methods and checking a cache directory (defined by environment variable PY2TO3_CACHEDIR) for previous conversion attempts for this file.

If the md5sum for the source matches the original conversion attempt, the original results are returned greatly speeding up repeated 2to3 conversions.

If the md5sum doesn’t match, it does the 2to3 conversion and than stashes a copy of the results into the cache dir for future usage.
