eclasses¶

object implementing the descriptor protocol for use in Just In Time access to attributes.

Consumers should likely be using the jit_func() line of helper functions instead of directly consuming this.

get_eclass(eclass)[source]¶
get_eclass_data(inherits)[source]¶

Return the cachable entries from a list of inherited eclasses.

Only make get_eclass_data calls for data you know came from this eclass_cache, otherwise be ready to catch a KeyError exception for any eclass that was requested, but not known to this cache.

is_eclass_data_valid(ec_dict)[source]¶

Check if eclass data is still valid.

Given a dict as returned by get_eclass_data, walk it comparing it to internal eclass view.

Returns:a boolean representing whether that eclass data is still up to date, or not
