class pkgcore.cache.metadata.database(location, *args, **config)[source]¶

Bases: pkgcore.cache.flat_hash.database

Compatibility with (older) portage-generated caches.

Autodetects per entry if it is a flat_hash.database or PMS compliant cache entry, and converts old (and incomplete) INHERITED field to _eclasses_ as required.

class pkgcore.cache.metadata.paludis_flat_list(*args, **config)[source]¶

Bases: pkgcore.cache.metadata.database

(Hopefully) write a paludis specific form of flat_list format cache. Not very well tested.

difference from a normal flat_list cache is that mtime is set to ebuild for normal, for paludis it’s max mtime of eclasses/ebuild involved.
