snakeoil.descriptors.classproperty[source]¶

Like the builtin property() but takes a single classmethod.

Essentially, it allows you to use a property on a class itself- not just on it’s instances.

Used like this:

>>> from snakeoil.descriptors import classproperty
>>> class foo(object):
...
...   @classproperty
...   def test(cls):
...     print "invoked"
...     return True
>>> foo.test
invoked
True
>>> foo().test
invoked
True
