pkgcore.config.basics.convert_hybrid(central, value, arg_type)[source]¶

Automagically switch between L{convert_string} and L{convert_asis}.

L{convert_asis} is used for arg_type str and if value is not a basestring. L{convert_string} is used for the rest.

Be careful about handing in escaped strings: they are not unescaped (for arg_type str).
