.. _`pinspect pkgsets manpage`:

================
pinspect pkgsets
================

synopsis
========

pinspect pkgsets [-h] [--all | pkgset [pkgset ...]]

DESCRIPTION
===========

pkgset related introspection

positional arguments
====================

:pkgset:  pkgset to inspect

optional arguments
==================

-h, --help  show this help message and exit
--all       display info on all pkgsets
